Introduction

The Basic Programming Language is a simple programming language designed for learning the fundamentals of programming. It provides basic constructs for performing tasks and calculations.

Data Types

Basic Programming Language supports the following data types:

  • Integer: Whole numbers (e.g., 42)
  • Float: Numbers with decimal points (e.g., 3.14)
  • String: Text data (e.g., "Hello, World!")
  • Boolean: True or False
Variables

Variables are used to store data. They must be declared before use.

INTEGER age age = 25
Operators

Basic Programming Language supports the following operators:

  • Arithmetic: +, -, *, /, %

  • Comparison: =, <, >, <=, >=, <>

  • Logical: AND, OR, NOT

Control Structures

CONDITIONAL STATEMENTS

IF...THEN...ELSE
IF x > 10 THEN PRINT "x is greater than 10" ELSE PRINT "x is not greater than 10" END IF

Loops

FOR loop
FOR i = 1 TO 10 PRINT i NEXT i
Functions

Functions are used to group and reuse code.

FUNCTION greet(name) PRINT "Hello, " + name + "!" END FUNCTION greet("Alice")
Input and Output

PRINT is used to display output.

INPUT is used to accept input from the user.

Comments

Comments are ignored by the interpreter and are used for adding explanations to the code.

REM This is a comment
Example Program

Here's a simple program that calculates the square of a number:

INTEGER number, square INPUT "Enter a number: ", number square = number * number PRINT "The square of " + number + " is " + square

This technical documentation provides a basic overview of the Basic Programming Language. In practice, real programming languages have additional features, libraries, and capabilities to handle more complex tasks and applications.
